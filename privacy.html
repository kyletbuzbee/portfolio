Privacy Policy

Last Updated: October 31, 2025

1. Introduction

This Privacy Policy describes how we collect, use, process, and protect your information when you use our Google Cloud-based applications and services ("Services"). We are committed to ensuring the privacy and security of your data while leveraging Google Cloud Platform's enterprise-grade security infrastructure.

Important: This privacy policy complies with Google's privacy requirements, applicable data protection laws (GDPR, CCPA, etc.), and industry best practices for cloud-based applications.

2. Information We Collect

2.1 Personal Information

  • Account Information: Name, email address, phone number, company information

  • Profile Information: Preferences, settings, role assignments
  • Usage Data: Feature usage, interaction patterns, performance metrics

2.2 Technical Information

  • System Information: IP address, browser type, device information, operating system
  • Cloud Infrastructure Data: Resource usage, compute metrics, storage consumption
  • Security Logs: Access logs, security events, audit trails
  • Performance Data: Response times, error rates, system performance metrics

2.3 Third-Party Integration Data

  • Google Services: Google Workspace data, Drive files, Calendar events, Contacts
  • Third-Party APIs: Data from integrated external services (with user consent)
  • Analytics Data: Aggregated usage statistics and performance metrics

3. How We Use Your Information

3.1 Primary Purposes

  • Service Delivery: Providing, maintaining, and improving our cloud-based services
  • Account Management: User authentication, authorization, and account administration
  • Performance Monitoring: Ensuring service availability, performance, and reliability
  • Security: Detecting and preventing fraud, abuse, and security threats

3.2 Legal and Compliance

  • Regulatory Compliance: Meeting legal obligations and industry standards
  • Audit Trail: Maintaining records for compliance and legal requirements
  • Data Protection: Implementing appropriate security measures and privacy controls
Google Cloud Compliance: Our services are hosted on Google Cloud Platform and comply with Google's privacy policies, security standards, and data processing agreements.

4. Google Cloud Platform Data Processing

4.1 Data Storage and Processing

Your data is stored and processed on Google Cloud Platform infrastructure, which includes:

  • Google Cloud Storage: Secure object storage for files and backups
  • Cloud SQL: Managed database services with encryption at rest and in transit
  • Compute Engine: Virtual machines running our application services
  • BigQuery: Data analytics and business intelligence processing
  • Cloud Functions: Serverless computing for event-driven processing

4.2 Data Security on Google Cloud

  • Encryption: All data encrypted at rest using AES-256 and in transit using TLS 1.2+
  • Access Controls: Identity and Access Management (IAM) with principle of least privilege
  • Network Security: Virtual Private Cloud (VPC) with firewall rules and private networking
  • Monitoring: Cloud Security Command Center and Cloud Logging for security monitoring
  • Compliance: SOC 2, ISO 27001, HIPAA, and other industry certifications

5.1 We Do Not Sell Personal Information

5.3 Data Localization

Your data may be processed and stored in various Google Cloud regions based on your location and service requirements. We will inform you of any significant changes to data storage locations.

6. Data Retention

6.1 Retention Periods

  • Active Accounts: Data retained as long as your account is active
  • Deleted Accounts: Data deleted within 30-90 days of account deletion
  • Legal Requirements: Some data retained longer to comply with legal obligations
  • Backup Data: Automatically deleted from backup systems within 30 days

6.2 Data Deletion

Upon request or account deletion, we will:

  • Permanently delete your personal data from all systems

  • Provide confirmation of data deletion upon request
  • Retain only anonymized data for analytical purposes
